# Video Touch Enhancer

> 为触屏设备优化的网页播放器手势增强脚本 


## ✨ 手势操作
| 手势 | 动作 | 反馈 |
|------|------|------|
| **单击** | 显示/隐藏控制栏 | — |
| **双击** | 播放/暂停 | — |
| **长按** | 进入倍速播放 | 倍速闪烁动画 |
| **横向滑动** | 调节视频进度 | 显示当前时间 / 总时长 |
| **纵向滑动** | 调节亮度音量 | 显示亮度音量百分比 (0–100%) |


## 🧩 功能特性
- **可视化设置面板**：可通过 Tampermonkey 菜单打开设置面板，修改脚本参数。
- **锁定按钮**：可锁定当前手势区域，防止误触。
- **菜单按钮**：可展开快退 / 快进快捷按钮，点击后跳转视频进度。
- **音量增强**：支持将视频音量调节到 100% 以上。
- **亮度增强**：支持将视频亮度调节到 100% 以上。
- **手势操作自定义**：可开启或关闭双击、长按、横向滑动功能，并可设置纵向滑动功能。
- **按钮区域自定义**：可设置左右按钮功能、跳转时长。


## 📦 安装
1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 访问 [本脚本页面](https://greasyfork.org/zh-CN/scripts/584363)
3. 点击「安装此脚本」


## 💻 兼容性
- 适用于主流网页播放器
- 支持常见浏览器（Edge / Chrome 等）
- 适用于触屏设备（Surface / Yoga / MateBook 等）
- 同样支持鼠标操作
